# Grupo Baronne Transporte e Paletes

Site institucional profissional desenvolvido com Next.js, Tailwind CSS e Framer Motion.

## Como rodar

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Build de produção

```bash
npm run build
npm run start
```

## Imagens

Adicione os arquivos reais em `public/images` mantendo exatamente estes nomes:

- `logo.png`
- `paletes-parede.jpg`
- `caminhao-paletes.jpg`
- `galeria-paletes.jpg`

O código já referencia esses arquivos e usa fallbacks visuais enquanto eles não são adicionados.
