# Imagens do site

Adicione nesta pasta os arquivos finais da identidade visual do Grupo Baronne:

- `logo.png`: logotipo oficial em PNG, preferencialmente com fundo transparente.
- `paletes-parede.jpg`: imagem de paletes empilhados para a seção de destaque.
- `caminhao-paletes.jpg`: caminhão ou operação logística com paletes para o hero da home.
- `galeria-paletes.jpg`: imagem de galeria, fábrica, estoque ou operação com paletes.

Mantenha exatamente esses nomes para que o site carregue os assets automaticamente.
